## 🤔 What Is This Project?

**StegaGen Secure** is a professional audio steganography system that:

1. **Hides audio files inside images** using advanced steganography
2. **Uses Genetic Algorithms** to optimize where audio bits are hidden
3. **Encrypts with AES-256** before hiding for maximum security
4. **Measures quality** using PSNR, MSE, and SSIM metrics
5. **Provides beautiful UI** with React + TypeScript + Tailwind CSS

### Real-World Use Case
```
You have: secret-message.mp3 + vacation-photo.jpg
    ↓
System creates: innocent-looking.jpg (contains your audio!)
    ↓
Nobody knows there's audio hidden inside
    ↓
You can extract it later with your encryption key
```
