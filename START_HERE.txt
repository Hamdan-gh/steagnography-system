📋 QUICK START (3 Steps)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Start Backend (Python)
───────────────────────────────────────────────────────────────────────────────
Open a terminal and run:

    cd python-engine
    python app.py

Wait for this message:
    ====================================================
    Starting StegaGen Processing Engine
    Server: http://localhost:5000
    ====================================================

✅ KEEP THIS TERMINAL OPEN!


STEP 2: Start Frontend (React)
───────────────────────────────────────────────────────────────────────────────
Open a NEW terminal and run:

    npm run dev

Browser will open: http://localhost:5173


STEP 3: Test It!
───────────────────────────────────────────────────────────────────────────────
1. Go to "Embed Audio" page
2. Check green "Backend Online" status (top-right)
3. Upload:
   - Image: Any PNG/JPG (500×500 to 1000×1000 for testing)
   - Audio: Any MP3/WAV (< 500KB for quick test)
4. Click "🔐 Auto Generate" for encryption key
5. Click "⚡ Quick Mode" preset button
6. Click "▶ Start Embedding"
7. Wait 30-90 seconds (watch the progress bar!)
8. Click "📥 Download" when complete

✅ DONE! Your audio is now hidden in the image!

🔍 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Backend Offline" status
Solution: Run: cd python-engine && python app.py

Problem: Still stuck at Step 5
Solution: 
  1. Use smaller files (< 1MB image, < 500KB audio)
  2. Click "⚡ Quick Mode"
  3. Or set: Generations=10, Population=10

Problem: "Audio file too large"
Solution: Use larger image OR smaller audio
  Formula: Image capacity = (width × height × 3) / 8 bytes
  Example: 1000×1000 image = 375KB capacity

Problem: Low quality (PSNR < 35)
Solution: Increase Generations to 50-100, Population to 30-50
